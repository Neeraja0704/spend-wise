## 💡 Quick Reference

### File Locations
- **Components:** `src/components/`
- **Pages:** `src/pages/`
- **Styles:** `src/App.css`, `src/styles/theme.css`
- **HTML:** `public/index.html`

### Key Concepts
- **State Management:** localStorage + React hooks
- **Routing:** In App.js with simple page switching
- **Styling:** CSS3 with color variables (theme.css)
- **Architecture:** Component-based, data flows from App

### Common Commands
```bash
npm install          # Install dependencies
npm start            # Start dev server
npm run build        # Production build
npm test             # Run tests
```

## 🚫 Common Mistakes

❌ **Mistake:** Trying to run without Node.js
✅ **Fix:** Install Node.js from nodejs.org first

❌ **Mistake:** Running from wrong folder
✅ **Fix:** Make sure you're inside the `expense-tracker` folder

❌ **Mistake:** Not waiting for `npm install` to finish
✅ **Fix:** Wait until you see the prompt $ again

❌ **Mistake:** Changing code but not seeing changes
✅ **Fix:** Did you save the file? Chrome auto-refresh should work.

❌ **Mistake:** "Port already in use" error
✅ **Fix:** See [SETUP_GUIDE.md](SETUP_GUIDE.md#issue-port-3000-already-in-use)
